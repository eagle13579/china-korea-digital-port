### 2.2 痛点 Top 3

| 优先级 | 痛点 | 详细描述 | 频率 |
|:------:|:-----|:---------|:----:|
| P1 | **「中国合规复杂得让人无从下手」** | 外商投资法、数据安全法、网络安全法、行业准入负面清单...政策文件几十份，不知道从哪看起。找韩国律所咨询太贵（起步¥10万+），找中国代理怕不靠谱。 | 每天 |
| P2 | **「找人太难，找到对的人更难」** | 想找一个懂韩语+懂中国合规+靠谱的人。市面上要么是韩国派遣的高薪外派员，要么是不会韩语的中国代理。沟通成本极高，信任建立慢。 | 每周 |
| P3 | **「想要快速验证中国市场，不想花半年走流程」** | 韩国市场卷不动了想进中国，但担心：花3个月了解政策→花3个月找代理→花3个月办执照→花3个月招人→1年过去什么都没卖出去。想要一个能「几天出方案、几周上轨道」的服务。 | 决策时 |

### 2.3 决策逻辑

```
信息获取链路:
Naver/Google搜索"중국 진출" → 看KOTRA免费资料 → 对比几家咨询公司报价
→ 问同行/韩国贸易协会推荐 → 找到1-2个候选 → 比价+比方案 → 决策

决策链:
CEO本人（决策人） ← 可能有1-2名韩国副总（参考意见） ← 无中方团队

决策关键因素（按权重）:
1. 信任感（40%） — 有没有韩国人做过？有没有韩国案例？有没有韩语服务？
2. 时间和效率（30%） — 能不能快速出方案？能不能3个月内开始卖货？
3. 价格（20%） — 首年预算¥5-20万，能用AI低成本完成初筛特别好
4. 品牌形象（10%） — 在中国做业务需要体面，不是随便找个小代理

信任 > 速度 > 价格 > 品牌
```

### 2.4 价格敏感度

| 维度 | 敏感度 | 说明 |
|:-----|:-------|:-----|
| **对¥0-¥10,000的服务** | 低 | 免费初评/深度方案¥9,800很容易决策，几乎不需要高层审批 |
| **对¥10,000-¥60,000的服务** | 中 | 年订阅¥58,000需要CEO亲自拍板，但对标律所¥100,000+仍然很便宜 |

### 3.2 痛点 Top 3

|:------:|:-----|:---------|:----:|
| P1 | **「中国合规政策变得太快，跟不上」** | 《数据安全法》实施后数据跨境审查趋严、新《公司法》2024修订后注册资本5年实缴、各地环保/安监标准不一致...韩国总部不理解为什么每年合规成本都在涨。 | 每月 |

| P3 | **「找专业服务要么太贵要么不靠谱」** | 四大的合规审计¥30万起，国内律所虽然便宜但不懂韩企文化和沟通方式（需要反复解释韩国总部的意图）。市面缺乏「懂韩企+懂合规+价格合理」的中间选项。 | 持续 |

### 3.3 决策逻辑

```
信息获取链路:
韩企微信群/在华韩国人圈子 → 中国合规峰会/研讨会 → 咨询同行好友
→ 韩国总部法务部意见 → 试用几款工具/服务 → 向总部汇报 → 决策

决策链:
中国法人代表（提议人+使用者） → 韩国总部法务部（审阅） → 韩国总部CFO（预算审批）

决策关键因素（按权重）:
1. 合规有效性（35%） — 能不能真的帮我避开合规风险？不能因为用了AI出问题
2. 成本可视化（30%） — 能不能给韩国总部一个清晰的成本收益报告？
3. 韩企理解度（20%） — 懂不懂韩国总部的决策逻辑和沟通方式？
4. 使用便捷性（15%） — 中文界面我能用，但韩国总部也要能看懂韩语报告

合规有效性 > 成本可视化 > 韩企理解度 > 便捷性
```

### 3.4 价格敏感度

| 维度 | 敏感度 | 说明 |
|:-----|:-------|:-----|
| **对¥0-¥10,000的服务** | 极低 | 法人代表可以直接决定，用于「试探性采购」——给总部看个结果再推动大预算 |
| **对¥10,000-¥60,000的服务** | 低 | 年订阅¥58,000对于在华子公司来说，是「请不起律所时的替代方案」 |

## 五、附录：韩企决策文化认知

### 5.1 韩企采购决策特征

1. **信任优先于产品** — 韩企用熟人介绍/推荐的比例极高。先建立信任关系，再谈采购。
2. **总部控制型决策** — 即使是在华法人代表，超过一定金额的采购必须韩国总部批准。需要准备好韩语材料让法人代表「拿着去汇报」。
3. **价格不是第一要素** — 韩企更在意「花了钱能不能交差」。如果中韩港能帮法人代表在总部面前展示业绩，价格是次要的。
4. **终身学习焦虑** — 韩国企业家普遍有「不学习就跟不上」的焦虑。对AI新工具的接受度比想象中高，但需要有韩语的手把手教学。
5. **圈子文化** — 在华韩国人圈子（微信群/KakaoTalk群）是信息和口碑的核心传播渠道。一个满意客户可以带来5-10个推荐客户。

### 5.2 销售触点建议

| 触点 | 画像1（CEO） | 画像2（法人代表） |
|:-----|:-------------|:-----------------|
| **在线搜索** | Naver SEO（韩语关键词） | Naver/百度搜索合规新闻 |
| **社交渠道** | KakaoTalk、Instagram | 微信（在华韩人群）、LinkedIn |
| **线下渠道** | 韩国贸易协会活动 | 在华韩国商会、韩企校友会 |
| **推荐来源** | KOTRA、贸易协会 | 在华韩国同行、律所推荐 |
| **内容偏好** | 短视频（韩语案例演示） | 白皮书、合规指南（PDF可下载） |
