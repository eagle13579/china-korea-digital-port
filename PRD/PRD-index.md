## 战略基石 — 时光机理论

> 本文件及所有PRD均基于 **时光机理论战略框架.md**（另见独立文档）构建。
> 时光机六步法是中韩出海数智港产品开发的底层战略逻辑，所有PRD的立项、优先级排序和验收标准均围绕六步法展开。

### 时光机六步法（核心框架）

| 步骤 | 名称 | 在数智港中的落地含义 |
|:----:|:-----|:---------------------|
| **①** | **CFC（Copy From China）** | 拆为两条线：**标准CFC**（中企进韩国→复制中国成熟SaaS模式）& **反向CFC**（韩企进中国→复制韩国合规经验） |
| **②** | **DTC（Direct To Consumer）** | 绕过韩国传统IT渠道，直接触达韩企CEO/合规负责人，找真实需求 |
| **③** | **套利** | 三重结构性差距变现：**势能套利**（中国AI成本 vs 韩国AI成本 10-50倍差）、**平台套利**（韩国AI合规SaaS竞品空白）、**定价套利**（中国工程师 vs 韩国工程师 3-5倍差） |
| **④** | **降维** | 五种能力碾压：AI工程化速度、全栈产品力、合规知识库厚度、数字员工团队规模、中韩双文化理解 |
| **⑤** | **小闭环** | 三阶验证飞轮：信息（网站/SEO→线索）→ 交易（咨询/报价→成交）→ 增长（口碑/渠道→复购） |
| **⑥** | **大闭环** | 杠杆×题材×融资终局公式：AI合规SaaS平台 + 中韩跨境题材 + 规模化资本路径 |

### 数智港关键的时光机应用

- **CFC双轨制**: 标准CFC（中→韩）与反向CFC（韩→中）并存，两份市场洞察驱动产品双线并行
- **三重套利**已嵌入定价策略（PRD-004）和商业化路线图
- **小闭环改造**: 30天冲刺改造为三阶验证（信息阶→交易阶→增长阶），每阶有独立PRD门控

```
  【商羊覆盖】SEO词库+品牌一致性检查
  【英招覆盖】销售场景验证+话术测试

  【蠪侄覆盖】定价模型验证+成本预测
  — 本阶段对应时光机【② DTC + ③ 套利】— 通过信息阶触达真实需求并验证定价套利假设

Phase 2 (1-2周): AI合规自检工具 + 支付 + 部署上线
  — 本阶段对应时光机【③ 套利 + ④ 降维 + ⑤ 小闭环】— 平台套利落地+能力碾压+交易阶闭环

Phase 3 (1-3月): 数字员工在线对话 + 微信小程序 + SEO内容
  — 本阶段对应时光机【⑤ 小闭环（增长阶）+ ⑥ 大闭环】— 从交易到增长再到规模化
```

| **PRD-005** 表单修复 | **⑤ 小闭环（信息阶）** | 表单修复确保信息收集通道畅通，是小闭环信息阶的基础设施 |
| **PRD-006** 服务咨询全流程 | **⑤ 小闭环（交易阶）** | 咨询→报价→成交的完整漏斗，是小闭环从信息阶到交易阶的核心管道 |
| **PRD-007** 合规自检工具 | **③ 套利（平台套利）+ ④ 降维** | 韩国AI合规SaaS竞品空白，8问答→PDF报告是平台套利和合规知识碾压的体现 |
| **PRD-008** 演示预约系统 | **⑤ 小闭环（交易阶）** | 演示预约是交易阶的关键转化节点，连接信息阶获取的线索与成交 |
| **PRD-009** KOI/BD渠道对接 | **⑤ 小闭环（增长阶）** | 渠道对接收口从交易阶延伸至增长阶，启动口碑/渠道复购飞轮 |
