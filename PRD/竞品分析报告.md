## 一、分析概述

### 1.1 分析范围
本报告覆盖中韩出海数智港在「韩企进入中国市场」这一垂直赛道的竞品，从**企业服务/合规咨询/AI+合规**三个交叉维度进行扫描。竞品筛选标准：
- 提供中国企业合规/市场准入相关服务的机构
- 服务对象包含韩企或外资企业
- 在AI+合规领域有产品化动作

### 1.2 市场定位
中韩出海数智港定位为：**AI驱动的中韩双边跨境合规服务平台**，核心差异化是6位AI数字合规官覆盖全链路。当前阶段主要服务韩企进入中国。

## 二、竞品分析矩阵

### 竞品1: Deel

| 维度 | 内容 |
|:-----|:------|
| **公司名+定位** | Deel — 全球合规雇佣与支付平台。估值120亿美元（2024年），累计融资约6.79亿美元。以色列创始，总部旧金山。 |
| **核心产品/服务** | 全球EOR（名义雇主）、合规雇佣、薪酬管理、 Contractor管理。支持150+国家。2024年上线合规模块，提供AI驱动的合同审查和合规检查。 |
| **定价模式** | EOR $599/人/月起；合规模块 $299-599/月附加订阅。按人头+月费模式。 |
| **客户群体** | 全球科技公司、跨国企业、远程团队。韩国客户主要为首尔/京畿道科技企业。 |
| **中韩港差异化机会** | Deel的合规产品是通用型，**不专精中国合规**，更不聚焦中韩双边。中韩港可以提供更垂直（仅中韩）、更深度（6位数字员工）、更低价（¥9,800起）的方案。 |
| **威胁程度** | 中 — Deel品牌强、渠道广，但定价高、中国合规深度不足，对中韩港早期客户抢夺有限。 |

### 竞品2: 钛动科技 (Tec-Do)

| 维度 | 内容 |
|:-----|:------|
| **公司名+定位** | 钛动科技 — 中国企业出海数字化服务商。B轮融资，估值约20亿人民币。广州总部。 |
| **核心产品/服务** | 出海营销SaaS、海外广告投放、KOL营销、跨境支付。主要服务中国企业出海，非韩企进中国。 |
| **定价模式** | SaaS月费¥5,000-50,000+广告消耗。按效果付费+订阅混合。 |
| **客户群体** | 中国出海企业（游戏、电商、APP）。 |
| **中韩港差异化机会** | 钛动聚焦中国→海外，中韩港是**双向**（韩企→中国是核心差异）。且钛动不做合规，中韩港有合规护城河。 |
| **威胁程度** | 低 — 客户群体不重叠，方向相反。 |

### 竞品3: KOTRA (大韩贸易投资振兴公社)

| 维度 | 内容 |
|:-----|:------|
| **公司名+定位** | KOTRA — 韩国政府下属贸易投资促进机构。非营利性，全球127个贸易馆。每年预算约5,000亿韩元。 |
| **核心产品/服务** | 免费市场调研、贸易配对、Invest Korea投资咨询、海外贸易馆对接。2024年起探索AI贸易咨询工具。 |
| **定价模式** | 大部分服务免费（政府补贴）。部分深度咨询外包给合作律所/咨询公司。 |
| **客户群体** | 所有韩国中小/中坚/大企业。 |
| **中韩港差异化机会** | KOTRA是**半官方渠道**，免费但服务泛、深度不足、响应慢。中韩港的AI数字员工可以做到7x24h、深度定制、快速响应。更关键的是——中韩港可以作为KOTRA的**商业化补充服务商**（合作而非对抗）。 |
| **威胁程度** | 中 — KOTRA免费服务会分流预算敏感型客户，但其深度能力有限，可作为渠道合作伙伴。 |

### 竞品4: 韩国四大律所中国业务部 (金·张 / 太平洋 / 广场 / 世宗)

| 维度 | 内容 |
|:-----|:------|
| **公司名+定位** | 韩国顶级律所中国业务部 — 金&张法律事务所、太平洋律师事务所、广场律师事务所、世宗律师事务所。韩国Top 4律所，年收入均超1,000亿韩元。 |
| **核心产品/服务** | 中国外商投资法务、合资合同、并购、合规审查、争议解决。人工服务为主，部分引进AI辅助（合同审查AI）。 |
| **定价模式** | 按小时计费：合伙人¥3,000-5,000/小时，律师¥1,000-2,000/小时。单个合规项目通常¥100,000-500,000。 |
| **客户群体** | 大型韩企、财阀。 |
| **中韩港差异化机会** | 价格悬殊：中韩港的深度方案¥9,800，年订阅¥58,000 vs. 律所单次项目¥100,000+。中韩港可以服务**大型韩企的初期筛查/低成本合规诊断**，以及**中小韩企（律所不服务的客户）**。 |
| **威胁程度** | 中 — 对大型客户有品牌和信任优势，但价格极高，中端市场留白。 |

### 竞品5: 中国本地合规SaaS (法大大 / 每刻报销 / 甄零科技)

| 维度 | 内容 |
|:-----|:------|
| **公司名+定位** | 法大大（电子合同SaaS，D轮，估值约60亿）、每刻报销（费控SaaS，C轮）、甄零科技（合同管理SaaS，A+轮）。 |
| **核心产品/服务** | 电子合同/费控报销/合同生命周期管理。聚焦**企业内部管理流程数字化**。 |
| **定价模式** | ¥10,000-50,000/年SaaS订阅。按企业规模阶梯定价。 |
| **客户群体** | 中国本土企业，少数服务外企。 |
| **中韩港差异化机会** | 这些SaaS解决的是「内部流程」问题，而中韩港解决的是「中韩跨境合规全过程」——市场准入、数据合规、财税、用工等。属于不同赛道，但客户重叠（韩企进中国后也需要这些工具）。中韩港可作为**入口**，后续对接这些SaaS。 |
| **威胁程度** | 低 — 不直接竞争，有生态合作空间。 |

### 竞品6: 传统中韩咨询公司 (华韩咨询 / 博达纳 / 个人顾问)

| 维度 | 内容 |
|:-----|:------|
| **公司名+定位** | 中小型中韩双边咨询公司/个人顾问。多数为在华韩国人/中国朝鲜族创办。规模3-20人。无融资。 |
| **核心产品/服务** | 市场进入咨询、公司注册代办、翻译服务、对接资源。完全人工服务。 |
| **定价模式** | 项目制¥20,000-80,000。无标准化产品，价格弹性大。 |
| **客户群体** | 中小韩企首次进入中国。 |
| **中韩港差异化机会** | 传统咨询高度依赖个人经验，质量不稳定、无工具化/产品化能力。中韩港的AI数字员工提供**标准化、可复制、7x24h**的服务，边际成本趋近于零。典型「AI替代传统咨询」场景。 |
| **威胁程度** | 低 — 传统咨询正在被AI替代，是中韩港直接颠覆的对象。 |

### 竞品7: 中国法律科技平台 (慧律 / 秘塔AI / 智合)

| 维度 | 内容 |
|:-----|:------|
| **公司名+定位** | 慧律（法律AI，A轮）、秘塔AI（法律搜索+合同AI，获红杉投资）、智合（法律媒体+培训）。 |
| **核心产品/服务** | AI法律检索、合同审查AI、法律知识库。聚焦**中国法律通用场景**。 |
| **定价模式** | ¥1,000-10,000/年订阅或按次付费。 |
| **客户群体** | 中国律师、法务。 |
| **中韩港差异化机会** | 这些平台解决通用中国法律问题，但**缺乏韩国视角**——不懂韩语、不懂韩国商业习惯、不懂中韩双边政策。中韩港的差异化在于「中韩双语+韩国人设计体验+韩企合规场景」。 |
| **威胁程度** | 低 — 场景不重叠，但长期可能向上延伸至跨境。 |

## 三、竞争格局总览

### 3.1 竞品定位地图

```
                    高价格
                      ↑
              四大律所中国部 ●
                      |
      Deel ●          |          传统中韩咨询 ●
                      |
     KOTRA(免费) ●    |          法律科技AI ●
                      |
            中国合规SaaS ●      钛动科技 ●
                      |
                    低价格
← 通用平台 -------- 中韩垂直 -------- 中国本土 →
```

### 3.2 中韩港的竞争护城河

| 护城河 | 说明 | 壁垒强度 |
|:-------|:------|:--------:|
| **中韩双语+垂直场景** | 韩国人设计、韩语母语体验、专注中韩跨境 | 高 |
| **6位数字合规官** | 全链路合规覆盖，AI驱动，7x24h | 高 |
| **极致性价比** | 对比律所/咨询公司 1/10~1/50价格 | 中 |
| **OPC+AI模式** | 超级个体+AI，轻团队重产品 | 中 |
| **先发优势** | 中韩AI合规垂直赛道目前空白 | 中（窗口期约6-12个月） |

### 3.3 威胁等级排序

| 威胁等级 | 竞品 | 理由 |
|:---------|:-----|:------|
| **高** | — | 目前无直接竞品占据"中韩AI合规"生态位 |
| **中** | Deel、KOTRA、四大律所 | 品牌/渠道/信任优势，但定位不同或价格过高 |
| **低** | 钛动科技、中国合规SaaS、法律科技、传统咨询 | 方向不同或即将被替代 |

## 四、中韩港差异化机会总结

### 4.1 核心差异化定位
> **「中韩跨境合规的AI数字员工平台」**
> - 不是通用合规SaaS（太泛）
> - 不是传统咨询公司（太贵）
> - 不是中国企业出海工具（反向）
> - 而是：**唯一专注韩企进中国的AI合规助手**

### 4.2 可进攻的市场空白

1. **韩企合规的「初筛-诊断」环节** — 律所不做、KOTRA做不深、AI能做
2. **中小韩企（律所不服务）** — 传统咨询服务的客户，价格敏感，AI替代
3. **大型韩企的「采购前自检」** — 在找律所之前先用AI扫一遍
4. **KOTRA渠道的AI增值层** — 作为KOTRA推荐的技术合作方
5. **韩商会的企业服务入口** — 在韩商会/协会场景中嵌入AI合规工具

### 4.3 威胁应对策略

| 威胁来源 | 应对策略 |
|:---------|:---------|
| Deel推中国合规模块 | 深耕中韩垂直场景，建立韩国社区口碑 |
| KOTRA自建AI工具 | 主动与KOTRA合作，成为其AI技术供应商 |
| 四大律所降价/推AI | 专注于律所不服务的「初筛」环节，做分发入口 |
| 中国法律科技平台跨境化 | 利用中韩双语+场景优势，建立语言/文化壁垒 |

## 五、合作与竞合建议

### 5.1 建议合作方

| 合作方 | 合作模式 | 价值 |
|:-------|:---------|:-----|
| **KOTRA** | 成为KOTRA推荐的AI合规服务商，嵌入KOTRA中国贸易馆服务流程 | 信任背书+获客渠道 |
| **韩国四大律所** | 中韩港做前端AI筛查，律所做后端深度服务，利润分成 | 客户导流+专业背书 |
| **韩人会/韩商会** | 在商会活动中演示/嵌入AI合规工具 | 精准获客 |
| **在华韩国创业园区** | 与韩国产业园（如Korea Startup Center）合作 | 早期客户孵化 |

### 5.2 不建议正面竞争的领域
- 大型韩企的全案法务外包（律所主战场）
- 免费政府咨询服务（KOTRA主战场）
- 中国本土企业内部管理SaaS（法大大等主战场）
